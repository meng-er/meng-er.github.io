mysql常用指令备忘

Posted on 2023-01-05
  • 登陆

mysql -u root -p;

密码gymA20000925

  • 查看数据库

show databases;

  • 新建数据库

    create database 数据库名;

    如新建名字为node_project的数据库

    create database node_project;

  • 新建表

    use database_name

    create table table_name(字段名 字段类型(长度),字段名 字段类型(长度)...);

如在node_project的数据库下新建一个含name和password的表

  • 删除数据库

drop database 数据库名;

提醒：删除操作不可撤回，数据库中的表和数据都会被删除，请勿轻易执行

  • 通过脚本执行sql操作

    新建sql脚本文件，例如init.sql，在该文件的目录下输入

    mysql -uroot -p密码<init.sql
